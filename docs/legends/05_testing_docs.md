# Легенда 5: Тестирование, документация и подготовка к защите

## Роль

Участник отвечал за проверку качества проекта, документацию, QA-материалы, презентационные материалы и подготовку объяснения архитектуры для защиты.

## Что сделал

- Поддерживал README в актуальном состоянии: запуск, переменные окружения, API, AI-импорт, OCR и сценарий демонстрации.
- Подготовил QA-документ с ответами на типовые вопросы по проекту.
- Описал обоснование технологий: backend, frontend, БД, OCR, LLM, Docker и тестирование.
- Подготовил легенды участников проекта.
- Проверял, что документация совпадает с текущими таблицами БД, API-эндпоинтами и интерфейсом.
- Поддерживал backend-тесты для авторизации, профиля, операций, статистики и импортов.
- Проверял frontend-сборку после крупных изменений интерфейса.
- Подготовил UML/архитектурные схемы и презентационные слайды по архитектуре, AI-импорту, выбору модели и производительности.
- Провел базовое нагрузочное тестирование AI-импорта в режимах текст, CSV и изображение.

## Что может рассказать на защите

Качество проекта обеспечивается не только тестами, но и понятной архитектурной документацией. Документы помогают объяснить, почему AI-импорт сделан через кандидатов, зачем нужен OCR, почему используется PostgreSQL и как пользователь контролирует финальную запись операций.

Также можно объяснить, что тесты используют mock LLM: это делает CI/локальный прогон стабильным, а реальную модель проверяют отдельно, потому что она зависит от Ollama, железа и установленного веса.

## Ответственность за качество

Проверял, что после изменений не расходятся код, README, QA, технологическое обоснование, легенды и презентация. Следил, чтобы демонстрационный сценарий был понятен человеку, который впервые открывает проект.

## Возможный вопрос на защите

**Что именно проверяют тесты?**

Тесты проверяют healthcheck, регистрацию, вход, обновление профиля с текущим паролем, конфликты email/логина, CRUD операций, user scope, статистику, текстовый импорт, CSV-импорт, image-импорт с mock OCR, подтверждение кандидатов, дубли и отдельные инструменты агента.
