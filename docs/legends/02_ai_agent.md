# Легенда 2: AI-агент и импорт банковских данных

## Роль

Участник отвечал за агентный пайплайн: извлечение операций из текста, CSV и изображений.

## Что сделал

- Реализовал общий `AgentPipeline`.
- Подключил mock-парсер для тестов и локальную Qwen-модель через Ollama для реального парсинга.
- Добавил structured output по JSON Schema, чтобы LLM возвращала не свободный текст, а строгий JSON.
- Собрал цепочку для картинок: изображение -> OCR -> нормализация текста -> LLM-парсинг.
- Добавил проверку дублей и нормализацию суммы, даты, типа операции и категории.

## Что может рассказать на защите

Главный акцент: AI не имеет права напрямую писать в финальные транзакции. Он создаёт кандидатов с confidence, а пользователь подтверждает или исправляет результат.

## Ответственность за качество

Проверял, что text-import и image-import проходят через один и тот же AI text parser, а тесты не зависят от медленной локальной модели.
